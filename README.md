# 🚨 SMS & Email Spam Detection System

An end-to-end Machine Learning web application built using **Python**, **Scikit-Learn**, and **Gradio** that classifies SMS or Email messages into **SPAM** or **HAM (Legitimate)**. 

The application is successfully deployed and hosted on [Hugging Face Spaces](https://huggingface.co/spaces/minhalali12/SMS-Detection).

---

## 🚀 Live Demo
You can try the live web interface directly here: [SMS Detection on Hugging Face](https://huggingface.co/spaces/minhalali12/SMS-Detection)

---

## 🛠️ Tech Stack & Libraries
* **Frontend/UI:** Gradio
* **Machine Learning:** Scikit-Learn (Multinomial Naive Bayes & TF-IDF Vectorizer)
* **Natural Language Processing (NLP):** NLTK (Tokenization, Stemming, Stopwords removal)
* **Model Deployment:** Hugging Face Spaces
